27 @section intro_sec Introduction
28 PyClustering is an open source data mining library written in Python and C++ that provides a wide range of clustering
29 algorithms and methods, including bio-inspired oscillatory networks. PyClustering is mostly focused on cluster analysis
30 to make it more accessible and understandable for users. The library is distributed under GNU Public License and
31 provides a comprehensive interface that makes it easy to use in every project.
32 
33 By default, the C++ part of the library is used for processing in order to achieve maximum performance. This is
34 especially relevant for algorithms that are based on oscillatory networks, whose dynamics are governed by a system of
35 differential equations. If support for a C++ compiler is not detected, PyClustering falls back to pure Python
36 implementations of all kernels.
37 
38 PyClustering consists of five general modules.
39

140 @section example_sec Examples
141 
142 The library provides intuitive and friendly interface. Here is an example how to perform cluster analysis using BIRCH
143 algorithm:
144 @code{.py}
145  from pyclustering.cluster import cluster_visualizer
146  from pyclustering.cluster.birch import birch
147  from pyclustering.samples.definitions import FCPS_SAMPLES
148  from pyclustering.utils import read_sample
149 
150  # Load data for cluster analysis - 'Lsun' sample.
151  sample = read_sample(FCPS_SAMPLES.SAMPLE_LSUN)
152 
153  # Create BIRCH algorithm to allocate three clusters.
154  birch_instance = birch(sample, 3)
155 
156  # Run cluster analysis.
157  birch_instance.process()
158 
159  # Get allocated clusters.
160  clusters = birch_instance.get_clusters()
161 
162  # Visualize obtained clusters.
163  visualizer = cluster_visualizer()
164  visualizer.append_clusters(clusters, sample)
165  visualizer.show()
166 @endcode
167 
168 Here is an how to perform cluster analysis using well-known K-Means algorithm:
169 @code{.py}
170  from pyclustering.cluster.kmeans import kmeans, kmeans_visualizer
171  from pyclustering.cluster.center_initializer import kmeans_plusplus_initializer
172  from pyclustering.samples.definitions import FCPS_SAMPLES
173  from pyclustering.utils import read_sample
174 
175  # Load list of points for cluster analysis.
176  sample = read_sample(FCPS_SAMPLES.SAMPLE_TWO_DIAMONDS)
177 
178  # Prepare initial centers using K-Means++ method.
179  initial_centers = kmeans_plusplus_initializer(sample, 2).initialize()
180 
181  # Create instance of K-Means algorithm with prepared centers.
182  kmeans_instance = kmeans(sample, initial_centers)
183 
184  # Run cluster analysis and obtain results.
185  kmeans_instance.process()
186  clusters = kmeans_instance.get_clusters()
187  final_centers = kmeans_instance.get_centers()
188 
189  # Visualize obtained results
190  kmeans_visualizer.show_clusters(sample, clusters, final_centers)
191 @endcode

196 An example of Hodgkin-Huxley oscillatory network simulation with 6 oscillators. The first two oscillators
197 have the same stimulus, as well as the third and fourth oscillators and the last two. Thus three synchronous
198 ensembles are expected after simulation.
199 @code{.py}
200  from pyclustering.nnet.hhn import hhn_network, hhn_parameters
201  from pyclustering.nnet.dynamic_visualizer import dynamic_visualizer
202 
203  # Change period of time when high strength value of synaptic connection exists from CN2 to PN.
204  params = hhn_parameters()
205  params.deltah = 400
206 
207  # Create Hodgkin-Huxley oscillatory network with stimulus.
208  net = hhn_network(6, [0, 0, 25, 25, 47, 47], params)
209 
210  # Simulate network.
211  (t, dyn_peripheral, dyn_central) = net.simulate(2400, 600)
212 
213  # Visualize network's output (membrane potential of peripheral and central neurons).
214  amount_canvases = 6 + 2 # 6 peripheral oscillator + 2 central elements
215  visualizer = dynamic_visualizer(amount_canvases, x_title="Time", y_title="V", y_labels=False)
216  visualizer.append_dynamics(t, dyn_peripheral, 0, True)
217  visualizer.append_dynamics(t, dyn_central, amount_canvases - 2, True)
218  visualizer.show()
