26 @section intro_sec Introduction
27 PyClustering library is a collection of cluster analysis, graph coloring, travelling salesman problem algorithms, oscillatory and neural network models, containers,
28 tools for visualization and result analysis, etc. High performance is ensured by CCORE library that is a part of the pyclustering library where almost the same algorithms,
29 models, tools are implemented. There is ability to use python code implementation only or CCORE (C/C++) implementation using special flag. CCORE library does not
30 use python.h interface to communicate with python code due to requirement to save ability to use CCORE library or C/C++ code part of CCORE in other projects.
31 
32 PyClustering consists of six general modules where the algorithms, models, tools are placed:
33

84 @section install_sec Installation
85 The simplest way to install pyclustering library is to use pip:
86 @code{.sh}
87  pip3 install pyclustering
88 @endcode
89 
90 The library can be compiled and manually installed on linux machine wherever you want:
91 @code{.sh}
92  # extract content of the pyclustering library...
93  # compile CCORE library (core of the pyclustering library).
94  cd pyclustering/ccore
95  make ccore
96 
97  # return to parent folder of the pyclustering library
98  cd ../
99 
100  # add current folder to python path
101  PYTHONPATH=`pwd`
102  export PYTHONPATH=${PYTHONPATH}
103 @endcode
104

109 @section example_sec Examples
110 
111 The library provides intuitive and friendly interface, cluster analysis can be easily performed:
112 @code{.py}
113  # an example of clustering by BIRCH algorithm.
114  from pyclustering.cluster.birch import birch;
115 
116  from pyclustering.utils import read_sample;
117 
118  # load data from the FCPS set that is provided by the library.
119  sample = read_sample(FCPS_SAMPLES.SAMPLE_LSUN);
120 
121  # create BIRCH algorithm for allocation three objects.
122  birch_instance = birch(sample, 3);
123 
124  # start processing - cluster analysis of the input data.
125  birch_instance.process();
126 
127  # allocate clusters.
128  clusters = birch_instance.get_clusters();
129 
130  # visualize obtained clusters.
131  visualizer = cluster_visualizer();
132  visualizer.append_clusters(clusters, sample);
133  visualizer.show();
134 @endcode
135 
136 Clustering algorithms can be used for image processing:
137 @code{.py}
138  # an example of image color segmentation.
139  from pyclustering.utils import draw_image_mask_segments, read_image;
140 
141  from pyclustering.samples.definitions import IMAGE_SIMPLE_SAMPLES;
142 
143  from pyclustering.cluster.kmeans import kmeans;
144 
145  # load image from the pyclustering collection.
146  data = read_image(IMAGE_SIMPLE_SAMPLES.IMAGE_SIMPLE_BEACH);
147 
148  # set initial centers for K-Means algorithm.
149  start_centers = [ [153, 217, 234, 128], [0, 162, 232, 128], [34, 177, 76, 128], [255, 242, 0, 128] ];
150 
151  # create K-Means algorithm instance.
152  kmeans_instance = kmeans(data, start_centers);
153 
154  # start processing.
155  kmeans_instance.process();
156 
157  # obtain clusters that are considered as segments.
158  segments = kmeans_instance.get_clusters();
159 
160  # show image segmentation results.
161  draw_image_mask_segments(IMAGE_SIMPLE_SAMPLES.IMAGE_SIMPLE_BEACH, segments);
162 @endcode

165 @image html fcps_cluster_analysis.png

167 Simulation of oscillatory network based on Hodgkin-Huxley neuron model where six synchronous ensembles of oscillators are formed. It means that
168 three features from input data are allocated where each feature is encoded by only one ensemble.
169 @code
170  # an example of simulation of oscillatory network based on Hodgkin-Huxley model
171  from pyclustering.utils import draw_dynamics;
172 
173  from pyclustering.nnet.hhn import hhn_network, hhn_parameters;
174 
175  # set period of 400 time units when high strength value of synaptic connection exists from CN2 to PN.
176  params = hhn_parameters();
177  params.deltah = 400;
178 
179  # prepare external stimulus that encode three different features.
180  stimulus = [0, 0, 25, 25, 47, 47];
181 
182  # create oscillatory network that has six oscillators.
183  net = hhn_network(len(stimulus), stimulus, params);
184 
185  # perform simulation during 1200 steps in 600 time units.
186  (t, dyn) = net.simulate(1200, 600);
187 
188  # visualize results of simulation (output dynamic of the network).
189  draw_dynamics(t, dyn, x_title = "Time", y_title = "V", separate = True);
190 @endcode
